# 🔍 AML Monitoring using Knowledge Graph

A web-based prototype that demonstrates how graph-based systems detect complex money laundering patterns.

---

## 🚀 Live Demo
https://lpg7728lpg-rgb.github.io/Aml-knowledge-graph/

---

## 🧠 Problem

Traditional AML systems use rule-based and linear monitoring, which fails to detect:
- Circular transactions
- Layered fund transfers
- Hidden relationships between accounts

---

## 💡 Solution

This project models financial transactions as a graph:

- Nodes → Accounts (A, B, C, D)
- Edges → Transactions

This allows detection of suspicious patterns like loops.

---

## ⚙️ Features

- Interactive graph visualization  
- Transaction simulation (A → B → C → D → A)  
- Cycle detection (fraud detection)  
- Risk scoring (Low / Medium / High)  
- Transaction logs  

---

## 🧪 How to Use

1. Open the live demo  
2. Click "Run Simulation"  
3. Observe transaction flow  
4. Suspicious loop will be highlighted  
5. Risk score increases  

---

## 🛠️ Tech Stack

## 📌 Key Insight
Money laundering is not linear — it is a network problem.
